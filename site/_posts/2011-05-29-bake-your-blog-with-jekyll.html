I've recently moved beyond using wordpress and jumped into the world of baked (or static) blogs. So how do you 'bake' your blog? This post is a look at building this blog using the open source static site generator, Jekyll.

-- Brent Simmons

"Baked pages are easy to serve. You can almost always switch servers and software and they’ll still work."
-- Aaron Swartz

Baked Blogs?

Since when did this become about cooking? It isn't. The idea is simple: Build your site in pure HTML/CSS/Javascript and let it live on the server as static files. "What? Is it the 90s again?", I hear you asking. Yeah, it is a bit old school, but it's not a crazy idea. Static files can be served from any server and it satisfies that geeky part of you that wants to keep your blog in source control.
